# Math
数学函数、算法等，暂时正在完善底层的大整数类，准备开始高精度浮点数类

有bint,bint2类，bint为10-bit,bint2为2-bit  
其中bint已经完善，bint2待完善  
均内置FFT，并用压位和动态扩展实现，支持直接赋值或字符串赋值等  

---

部分算法有待完善（特别是除法）

- 乘法复杂度为O(nlog n)，实现了暴力，Karatsuba（暂未使用）以及FFT ，bint2乘法常数小于bint，快约25%~30%
- 除法复杂度为O(n log ^ 2 n) ，采用的牛顿迭代+FFT，能力和精力有限，暂且搁置  
- 取模，和除法复杂度类似  

bint2一般来说比bint10快，因为采用了大量位运算，但考虑到十进制更为常用，且进制转化需要耗时，bint2暂时还不太完善

可支持快速2进制和10进制间的转换，复杂度 O ( n log n ) ，常数略大  
release版本*8e5位10进制转2进制需大约0.6~0.7s，8e6位需大约9.3s

注意！有部分地方重复代码是为了常数优化，例如可以使得编译器对其进行优化，或者使得可以进行位运算等！
